# 🏥 Hospital Meal Planner - Complete Ordering System

A modern hospital food ordering system with personalized meal planning, real-time order tracking, and kitchen management dashboard.

## ✨ Key Features

### For Patients
- 4-day meal plans with 2 swipeable options per meal
- One-click meal ordering
- Real-time order tracking with animated chef
- 30-minute countdown timer
- Visual progress timeline

### For Kitchen
- Centralized order management dashboard
- Accept → Prepare → Complete workflow
- Filter by status
- Auto-refresh every 5 seconds
- Color-coded order badges

### Medical Personalization
- Meals matched to 5 medical conditions
- Detailed nutritional information
- Health benefits explanations
- Dietary restrictions respected

## 🚀 Quick Start

**Run the app:**
```bash
python start.py
```

**Open browser:**
```
http://localhost:5001 (Patient Dashboard)
http://localhost:5001/kitchen (Kitchen Dashboard)
```

## 📖 How To Use

**Patients:**
1. Browse meal plan → Swipe between 2 options → Order meal
2. Go to "View My Orders" → See animated chef cooking
3. Watch 30-min countdown → Get notified when ready

**Kitchen:**
1. See new orders → Click "Accept"
2. Click "Start Cooking" → Status = In Progress  
3. Click "Mark as Done" → Patient sees "Ready!"
